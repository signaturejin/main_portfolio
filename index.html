의류 카테고리

- 의류를 등록할 때 상의 / 아우터 / 하의 / 신발 / 모자에 따라 분류하여 등록하였습니다.

- 데이터를 찾아올 때 각 카테고리에 맞는 데이터들을 매칭하여 불러오게 하였습니다.

매장검색기능

- 지역선택 부분에서 시/도를 선택하기 전에는 구/군에서 어떠한 지역도 뜨지 않습니다.

- 지역선택 부분에서 아무것도 정하지 않고 검색할 경우 모든 매장리스트가 나오도록 하였습니다.

- 해당 지역에 매장이 없을 경우 '매장 검색 결과가 없습니다.' 라는 문구가 뜨도록 하였습니다.

페이징기법

- 한 페이지당 보여줄 데이터수는 4개이며, 블록당 보여줄 페이징 번호수는 3개입니다.

- 1번/마지막 페이지를 보고 있을 경우 처음/마지막 버튼은 눌리지 않습니다.

- 이전/다음 버튼을 누를 경우 바로 이전/다음 페이지로 넘어갑니다.

- 처음/마지막 버튼을 누를 경우 바로 첫/마지막 페이지로 넘어갑니다.

관리자 등록페이지

- 관리자만 알 수 있는 의류 / 라이프 / 매장 등록페이지가 존재합니다.

- 등록할 때마다 데이터베이스에 데이터가 추가되며 실시간으로 페이지에도 추가한 데이터가 뜨게 됩니다.
